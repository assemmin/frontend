# AI Usage Report

## Tool Used
- **GitHub Copilot (Raptor mini Preview)**

## Prompts and Interactions
The assistant was used extensively to:
- Generate HTML, CSS, and JavaScript for the jQuery implementation.
- Scaffold, modify, and explain the React Vite projects for both labs.
- Produce README files and detailed comparison/explanation documents.
- Outline project structure and provide instructions for running code.
- Assist with Git initialization and conventional commit messages.
- Create explanatory text for comparison and view layer reports.

Each request from the user was followed by a corresponding response from the assistant, with code manually typed into the workspace.

## How Code Was Modified and Verified
- The assistant initially generated boilerplate code which was reviewed by me, the user, and then manually edited to match lab requirements.
- All code was manually verified by running the React development server (`npm run dev`) for both projects to ensure functionality.
- The jQuery imperative implementation was opened in a browser to verify the toggle behavior.
- CSS styles were tested visually in the browser to ensure highlighting and design.
- The written explanation texts were manually reviewed and edited for clarity, ensuring they met word count requirements.

## Learning and Validation
- This process reinforced understanding of imperative vs declarative UI paradigms.
- It provided hands-on experience using Vite to scaffold React projects and modify components.
- Through iterative editing and testing, I validated that state updates trigger React re-renders correctly and that event handlers work as expected.

I reviewed all generated code, made necessary edits, and confirmed correct behavior before submitting the lab deliverables.
